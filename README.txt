Introduction
============

This package makes it possible to use `zope.intid`_  and the `zope.app.intid`
Browserviews (and consequentually other packages that rely on it such as
``zope.keyreference``) in a Zope2 environment.

.. _zope.intid: http://pypi.python.org/pypi/zope.intid
.. _zope.app.intid: http://pypi.python.org/pypi/zope.app.intid
